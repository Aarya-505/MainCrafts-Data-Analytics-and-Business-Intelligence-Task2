## Project Overview
This repository contains the deliverables for Task 2 of the MainCrafts Technology internship. The objective was to perform an advanced end-to-end data analysis on raw Superstore sales data to uncover actionable business insights. The project spans initial data querying, exploratory data analysis, KPI calculation, and the development of a fully interactive, UI-optimized business intelligence dashboard.

## Tools & Technologies Used
* **Data Querying:** SQL (Relational JOIN queries)
* **Data Cleaning & EDA:** Microsoft Excel (Pivot Tables, Advanced logical formulas like SUMIFS/COUNTIFS)
* **Data Visualization:** Microsoft Power BI (DAX custom measures, Layered UI design, Interactive slicing)
* **Documentation:** Comprehensive Business Summary Report

## Key Deliverables Included in this Repository
* **Raw Data Files:** `train.csv` and associated dataset files containing the initial raw data.
* **SQL Queries:** `Task2_Queries.sql` containing the code used to query and merge tables.
* **Excel Analysis:** `Task2_Excel_Analysis.xlsx` containing data cleaning, pivot tables, and base KPI calculations.
* **Power BI Dashboard:** `Task2_Dashboard.pbix` (interactive file) and `Task2_Dashboard.pdf` (high-resolution snapshot) featuring custom navigation and business KPIs.
* **Business Summary Report:** `Business_Summary_Report.pdf` detailing a 3-page strategic summary on regional performance, category profitability, and customer segmentation.

## Quick Insights
* The **West** region is the primary revenue driver, accounting for 31.4% of total sales.
* The **Technology** category yields the highest absolute profit margins, whereas the Furniture category struggles with profitability, highlighting a critical need to reevaluate discount strategies.
* The **Consumer** segment is the most valuable demographic, driving over 50% of the total business revenue.
